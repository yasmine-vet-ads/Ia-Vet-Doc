# Problema

Na rotina veterinária, muitas informações importantes ficam distribuídas em documentos diferentes, como laudos, históricos clínicos, artigos científicos, protocolos, exames laboratoriais, imagens e materiais técnicos.

Essa fragmentação pode dificultar a recuperação rápida de informações relevantes, principalmente quando o profissional precisa revisar um caso, comparar achados, organizar dados clínicos ou consultar referências técnicas.

## Principais dificuldades observadas

- Informações clínicas espalhadas em diferentes documentos;
- dificuldade para localizar achados importantes rapidamente;
- ausência de padronização em alguns registros;
- necessidade de revisar manualmente grandes volumes de texto;
- risco de perda de contexto entre histórico, exame e conclusão;
- dificuldade para transformar documentos técnicos em dados organizados;
- tempo elevado para consultar informações recorrentes.

## Proposta do projeto

O projeto **Ia-Vet-Doc** propõe explorar o uso de IA para apoiar a organização, consulta e estruturação de documentos veterinários.

A ideia é criar um protótipo capaz de receber documentos-fonte, dividir o conteúdo em trechos, organizar as informações e permitir consultas assistidas por IA, sempre com revisão humana.

## Importante

Este projeto não tem como objetivo substituir a avaliação de um médico-veterinário.

A proposta é usar tecnologia como apoio à organização de informações, documentação técnica, consulta contextual e automação de processos.
